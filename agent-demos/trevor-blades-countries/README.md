# Trevor Blades Countries

A travel planner app powered by the Trevor Blades Countries GraphQL API. Explore countries with search and filtering, compare two countries side by side, and build a personalized travel itinerary.

![Screenshot](screenshot.png)

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy).

## Required Secrets

No secrets required for this project.

## Live Demo

<https://ivy-agent-demos-travel-planner.sliplane.app>

```
dotnet watch
```

ivy deploy
