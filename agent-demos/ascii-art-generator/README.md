# ASCII Art Generator

Convert text into ASCII art using a variety of FIGlet fonts, or upload images and transform them into ASCII character art with adjustable width and color inversion.

![Screenshot](screenshot.png)

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy).

## Required Secrets

No secrets required for this project.

## Live Demo

<https://ivy-agent-demos-ascii-art-generator.sliplane.app>

