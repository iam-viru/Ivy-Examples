# Cat Facts

A fun cat facts explorer featuring daily random facts from the CatFact API, a searchable breed catalog with pagination, and a favorites collection to save your best finds.

![Screenshot](screenshot.png)

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy).

## Required Secrets

No secrets required for this project.

## Live Demo

<https://ivy-agent-demos-cat-facts.sliplane.app>

## Run

```
dotnet watch
```

## Deploy

```
ivy deploy
```
