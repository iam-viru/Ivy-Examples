# Lovable Travel Booking

A travel booking application that lets users browse, search, and filter travel packages by category (Beach, Mountain, City, Adventure), view package details, and make bookings with a booking history tracker.

![Screenshot](screenshot.png)

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy).

## Required Secrets

No secrets required for this project.

## Live Demo

<https://ivy-agent-demos-lovable-travel-booking.sliplane.app>

## Run

```
dotnet watch
```

## Deploy

```
ivy deploy
```
