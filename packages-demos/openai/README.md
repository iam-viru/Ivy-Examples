# OpenAIExample 

Web application created using [Ivy](https://github.com/Ivy-Interactive/Ivy).

## Description

OpenAIExample demonstrates integration with OpenAI API for AI-powered features including chat, text generation, and natural language processing capabilities.

Ivy is a web framework for building interactive web applications using C# and .NET.
